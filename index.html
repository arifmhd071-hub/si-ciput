1. Buka console.firebase.google.com
2. Buat project → Firestore Database → mode test
3. Project Settings → Add Web App → copy config
4. Paste konfigurasi JSON di bawah ini

Fitur analisis AI menggunakan Google Gemini 2.0 Flash — gratis, tanpa kartu kredit.

🔑 Setup Google AI Studio

Fitur analisis AI menggunakan Google Gemini 2.0 Flash — gratis, tanpa kartu kredit.

Cara dapat API Key:

🔒API Key disimpan di perangkat ini saja. Tidak dikirim ke server manapun selain Google.
